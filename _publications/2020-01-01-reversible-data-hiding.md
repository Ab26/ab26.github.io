This paper presents a novel approach for reversible data hiding using multi-layer perceptron for pixel prediction. Reversible data hiding is a technique that allows the original cover media to be perfectly restored after the hidden data has been extracted. Our method uses a multi-layer perceptron neural network to predict pixel values, which improves the embedding capacity while maintaining high visual quality.
